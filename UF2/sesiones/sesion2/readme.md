2. Crea proyecto

4. Instala dependencia lite-server

6. Crea spa con los siguientes componentes:

     - Componente menu
   - Vista home
   - Vista about

8. Página partidas:

9. Instala la dependencia bootstrap (mira la documentación) y utiliza sus clases
