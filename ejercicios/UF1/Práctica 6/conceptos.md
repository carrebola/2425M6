## Funcionalitats JavaScript Utilitzades

### 1. **Math**
- `Math.random()`: Genera un número decimal aleatori entre 0 i 1.
- `Math.floor()`: Arrodoneix un número decimal cap a l'inferior més proper.
- **Ús**: Generació de números aleatoris i selecció d'elements aleatoris en arrays.

### 2. **Manipulació de l'arbre DOM**
- `document.querySelector()`: Selecciona el primer element del document que coincideix amb un selector CSS.
- `document.getElementById()`: Selecciona un element basant-se en el seu identificador únic (ID).
- `element.textContent`: Modifica o retorna el text que conté un element HTML.
- `element.innerHTML`: Modifica el contingut HTML d'un element.
- **Ús**: Per capturar, mostrar i actualitzar els valors de diferents elements del DOM com inputs, divs i botons.

### 3. **Manipulació d'entrades de text**
- `input.value`: Obté o estableix el valor d'un camp d'entrada (com un `input` o `textarea`).
- `.toUpperCase()`: Converteix tot el text d'una cadena a majúscules.
- `.toLowerCase()`: Converteix tot el text d'una cadena a minúscules.
- `.split()`: Divideix una cadena de text en un array de substrings basat en un separador.
- `.charAt()`: Retorna el caràcter situat en una posició específica dins d'una cadena.
- `.slice()`: Retalla una part d'una cadena, retornant una nova cadena amb els caràcters seleccionats.
- **Ús**: Per manipular el text de forma dinàmica, com convertir text a majúscules/minúscules, fer cerques i formatar dates.

### 4. **Estructures de control**
- `for` (bucle): Utilitzat per recórrer arrays o cadenes de text per realitzar operacions sobre cada element.
- `if-else`: Per a la presa de decisions basada en condicions.

### 5. **Manipulació d'arrays**
- `.push()`: Afegeix un element al final d'un array.
- `.length`: Retorna la longitud d'un array o cadena.

### 6. **Captura d'esdeveniments**
- `element.addEventListener()`: Afegeix un "listener" a un element per escoltar esdeveniments com el clic. Quan es detecta l'esdeveniment, s'executa una funció associada.
- **Ús**: Per gestionar la interacció de l'usuari, com els clics en botons per generar un resultat o modificar text.

### 7. **Concatenació de cadenes**
- `+=`: Utilitzat per construir cadenes de text mitjançant la concatenació de fragments individuals dins d'un bucle.
